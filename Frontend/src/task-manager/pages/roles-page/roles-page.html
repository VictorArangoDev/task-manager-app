System Role

Administrator

Full access to all project settings, billing, and team management.

  • Create Tasks & Subtasks

Read-only access to projects. Useful for clients or external stakeholders.

Project & Task Statuses

Pending

Default state for newly created tasks.

In Progress

Tasks currently being worked on by a team member.

Review

Waiting for approval or QA feedback.

Completed

Archived and successfully finished tasks.

On Hold

Blocked or temporarily suspended activities.
